Conway's Game of Life API Documentation

This is the API documentation for our group project on Conway’s Game of Life. The game executes Conway’s Game created by by the British mathematician John Horton Conway. You can click on the grids you want to fill or click on the random button to auto-fill, then click start game. All methods should be called on this object. Below you can find the API:

Method NameParametersReturn ValueDescription
randomize(cell)This creates random filled cells.NoneThis function receives a random value (stored in conway.howRandom) and activates random squares.
initialize()NoneNoneClears the grid and starts a new game.
gameLoop()Compares all the cells and implements Conway's game logic.NoneIterates through entire pixel 2d array checking and modifying the array.
setPixel()Toggles clicking on grid between True and FalseNoneThis enables mouse click on grid and flips true to false and vice versa causing the grid to be filled or not.
draw()Draws a 32 x 32 grid.NoneChecks each block for on/off state, fills in all true.
listen(cb)cb: is the callback function. This function will be called when the grid changes. The grid is passed to the callback functionNoneThe listen(cb) functionality is used to track change in the model and receive notifications. This allows the UI to be updated when the model changes.
